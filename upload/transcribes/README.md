# Примеры транскрибаций для IntuiUX Agent

Эта папка содержит примеры транскрибаций обсуждений идеи продукта для разных типов бизнесов. Каждый файл демонстрирует реалистичный диалог между заказчиком и специалистом.

---

## 📁 Структура папки

| Файл | Тип продукта | Отрасль |
|------|--------------|---------|
| `01-ecommerce-aquarium.md` | E-commerce | Аквариумистика / Зоотовары |
| `02-saas-taskflow.md` | SaaS | Продуктивность / Управление проектами |
| `03-b2b-logistics-crm.md` | B2B SaaS | Логистика / Транспорт |
| `04-blog-startups-media.md` | Медиа / Блог | Контент / EdTech |
| `05-landing-ux-course.md` | Landing Page | Образование |
| `06-dashboard-ecommerce-analytics.md` | Dashboard | E-commerce / Аналитика |
| `07-booking-nail-salon.md` | Booking / Онлайн-запись | Бьюти-индустрия |
| `08-app-fitness-tracker.md` | Mobile App | Фитнес / Здоровье |

---

## 🚀 Как использовать

1. Откройте [IntuiUX Agent](https://intuiux-agent.vercel.app)
2. Скопируйте содержимое любого файла транскрибации
3. Вставьте в поле ввода на этапе "Идея"
4. Нажмите "Отправить"
5. Agent проанализирует диалог и сформирует структурированную идею продукта

## 📋 Содержание каждого файла

Каждый файл включает:

- **Тип продукта** — классификация для IntuiUX
- **Отрасль** — сфера применения
- **Диалог** — реалистичная транскрибация обсуждения
- **Ключевые функции** — список возможностей продукта
- **Целевая аудитория** — сегменты пользователей
- **Монетизация / Тарифы** — бизнес-модель
- **Дифференциация** — преимущества перед конкурентами

## 🎯 Типы продуктов

### 1. E-commerce
Интернет-магазины с каталогом товаров, корзиной, оплатой и доставкой.

**Пример:** Интернет-магазин аквариумистики "Океан у дома"

**Ключевые слова для определения:**
- магазин, товар, каталог, корзина
- заказ, оплата, доставка
- карточки товаров, прайс-лист

---

### 2. SaaS
Облачные сервисы с подписочной моделью для команд и бизнеса.

**Пример:** Платформа управления проектами "TaskFlow"

**Ключевые слова для определения:**
- подписка, тариф, freemium
- команда, пользователи, collaboration
- интеграции, API, автоматизация

---

### 3. B2B
Продукты для бизнеса с длинным циклом сделки и корпоративными клиентами.

**Пример:** CRM для логистических компаний "LogiCRM"

**Ключевые слова для определения:**
- бизнес, корпоративный, enterprise
- интеграция с 1С, SSO
- продажи, менеджеры, отчёты

---

### 4. Blog / Медиа
Контентные платформы со статьями, видео, подкастами.

**Пример:** Медиа о стартапах "Стартаперы"

**Ключевые слова для определения:**
- статьи, блог, контент
- подписчики, рассылка
- SEO, реклама, премиум-контент

---

### 5. Landing Page
Одностраничные сайты для продукта, услуги или мероприятия.

**Пример:** Онлайн-курс по UX-дизайну

**Ключевые слова для определения:**
- курс, продукт, услуга
- регистрация, заявка
- отзывы, тарифы, CTA

---

### 6. Dashboard
Панели аналитики и мониторинга с визуализацией данных.

**Пример:** Аналитика для e-commerce

**Ключевые слова для определения:**
- метрики, графики, отчёты
- данные, аналитика, мониторинг
- KPI, dashboard, визуализация

---

### 7. Booking
Системы онлайн-записи и бронирования.

**Пример:** Онлайн-запись для студий маникюра

**Ключевые слова для определения:**
- запись, бронирование, расписание
- слоты, время, календарь
- напоминания, уведомления

---

### 8. Mobile App
Мобильные приложения для iOS и Android.

**Пример:** Фитнес-трекер "FitTrack"

**Ключевые слова для определения:**
- приложение, мобильное, iOS, Android
- пуш-уведомления, офлайн-режим
- App Store, Google Play

## 💡 Рекомендации по созданию транскрибаций

### Хороший диалог содержит:

1. **Контекст бизнеса** — чем занимается, сколько лет на рынке
2. **Проблему** — что болит, что не работает
3. **Целевую аудиторию** — кто клиенты, их потребности
4. **Конкурентов** — кто ещё на рынке, чем отличаются
5. **Функциональность** — что должен уметь продукт
6. **Ограничения** — сроки, бюджет, технические ограничения
7. **Дифференциацию** — почему выберут вас

### Формат диалога:

```
**Участник 1:** Вопрос или описание ситуации
**Участник 2:** Ответ, уточнение, предложение
```
